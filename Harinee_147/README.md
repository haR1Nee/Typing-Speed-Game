# Typing Speed Game

## Team Members
- Harinee
- Reena . H
- Nabirajan . P

## Description
A browser-based typing game where players type words quickly to score points before time runs out.

## How to Play
- Click Start
- Type the displayed word correctly
- Each correct word gives 10 points
- Game ends when timer reaches zero

## Concepts Used
HTML, CSS, JavaScript, DOM manipulation, events, loops, arrays, functions
