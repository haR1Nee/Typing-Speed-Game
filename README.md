# Typing Speed Game

## Project Overview
The Typing Speed Game is a browser-based interactive game developed using HTML, CSS, and JavaScript.
The objective is to type the displayed words as quickly and accurately as possible within a limited time.

---

## How to Play
1. Click the **Start Game** button.
2. A word will appear on the screen.
3. Type the word correctly in the input box.
4. Each correct word increases the score.
5. The game ends when the timer reaches zero.
6. Click **Replay** to play again.

---

## Features
- Start screen with instructions
- Real-time score tracking
- Countdown timer
- Game Over screen
- Replay option
- Interactive typing input

---

## Technologies Used
- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Handling
